# Employ-MGT

A Java-based **Employment Management System** designed to simplify employee administration for organizations. The application provides an efficient way to manage employee records, departments, and essential HR operations through a user-friendly interface.

## ✨ Features

- Employee registration and management
- Add, update, and delete employee records
- Department management
- Employee search functionality
- Employee information dashboard
- User-friendly interface
- Data persistence using a relational database



## 📌 Core Functionalities

- Employee Management
- Department Management
- Employee Record Maintenance
- Data Validation
- Search and Filtering
